## HANDSHAKE

### Pre requisites
1. NodeJS
2. Docker
### Get Started
1. **Run server app:**
  a. goto server dir: `cd server`
  b. Run docker for postgres and pgadmin: `npm run run:services`
  c. Run NestJS app: `npm run start:dev`

2. **Run client app:**
  a. goto client dir: `cd client`
  b. Run angular app: `npm start`
